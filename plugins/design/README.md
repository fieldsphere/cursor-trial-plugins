# Design

Focused plugin for UI/UX work:

- **wireframes** skill — Backend-aware wireframes and data states
- **mockup** skill — Production-oriented mockup workflow grounded in `design-schema.md`
- **designer** agent — Creative design direction, layout exploration, and visual critique
- **ui-engineer** agent — Technical UI implementation aligned with approved design intent
- **Figma MCP** in `mcp.json` when your team uses Figma

Plugin id: `**design`**. Display name: **Design**.
