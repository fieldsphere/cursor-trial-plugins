# Product Management

Focused plugin for product and delivery workflows:

- **write-ticket** — Implementation-ready tickets and work items
- **summarize-board** — Board, backlog, or sprint summaries
- **devils-advocate** — Challenges plans and assumptions before build
- **Atlassian MCP** in `mcp.json` as a default example integration for teams that use Jira/Confluence; replace or remove it for your stack

Plugin id: **`product-management`**. Display name: **Product Management**.
